
# File Spirit Matchmaker


File Spirit Matchmaker matches together related files based on file extentions.

Currently based on one or multiple source directories to be searched against one or multiple set of other directories.

This is also my learning project for python.

##  Common Use case
Sorting pictures, where images files *(eg. .jpg, .png)* may be seperated from raw *(eg. .Nef, .raw)* files for easier sorting and validation, and the sets are afterward to be moved to a seperate location.\
Instead of picking out each raw file speratly based on the remaining images files, the File Spirit Matchmaker picks out and moves the pair of files.
